# erp-portokalli Complete Feature Catalog

This document lists the business and platform features implemented in the `erp-portokalli` Laravel application.

The catalog was compiled from `routes/web.php`, `routes/tenant.php`, `routes/tenant-api.php`, `routes/inventory-procurement.php`, `routes/stock.php`, `routes/motorpool.php`, `routes/vehicle-management.php`, `routes/pos-api.php`, and the controller/service classes under `app/`.

## 17. Known Implementation Notes

| Topic | Note |
| --- | --- |
| Duplicated route blocks | Several legacy and SPA routes coexist for the same domain areas, especially customers, invoices, proposals, announcements, and HR modules |
| Route-list generation | `php artisan route:list` currently fails because `App\Http\Controllers\ConfigurationController` is referenced but not present in the expected file path, so this feature catalog was assembled by static route/controller review |
| Broad ERP scope | The application is not only inventory; it includes CRM, HR, payroll, accounting, contracts, certificates, projects, POS, reports, vehicle/motorpool, and portal modules |
